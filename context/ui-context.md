# UI Context

## Theme

Dark only. The UI uses near-black page backgrounds,
layered neutral surfaces, subtle borders, and high-contrast
text. The root document carries the `dark` class and
`globals.css` defines dark tokens as the default `:root`
values so no light theme appears by default.

## Colors

[Define your color tokens as CSS custom properties.
All components must use these tokens — no hardcoded
hex values.]

| Role            | CSS Variable       | Value    |
| --------------- | ------------------ | -------- |
| Page background | `--background`     | `oklch(0.145 0 0)` |
| Surface         | `--card`           | `oklch(0.205 0 0)` |
| Primary text    | `--foreground`     | `oklch(0.985 0 0)` |
| Muted text      | `--muted-foreground` | `oklch(0.708 0 0)` |
| Primary accent  | `--primary`        | `oklch(0.922 0 0)` |
| Border          | `--border`         | `oklch(1 0 0 / 10%)` |
| Error           | `--destructive`    | `oklch(0.704 0.191 22.216)` |
| Success         | Not defined yet    | Add when needed |

## Typography

| Role      | Font              | Variable      |
| --------- | ----------------- | ------------- |
| UI text   | Geist Sans        | `--font-sans` |
| Code/mono | Geist Mono        | `--font-mono` |

## Border Radius

| Context           | Class            |
| ----------------- | ---------------- |
| Inline / small UI | `rounded-sm` / `rounded-md` |
| Cards / panels    | `rounded-lg` |
| Modals / overlays | `rounded-xl` |

## Component Library

shadcn/ui on top of Tailwind CSS v4. Components live in
`components/ui/` and are generated by the shadcn CLI.
Do not hand-edit generated `components/ui/*` files after
installation.

Installed primitives:
- Button
- Card
- Dialog
- Input
- Tabs
- Textarea
- ScrollArea

## Layout Patterns

- Modals: centered overlay with subtle backdrop blur.
- Surfaces: use `bg-card`, `bg-popover`, `bg-muted`, and
  tokenized borders from `globals.css`.

## Icons

Lucide React. Use stroke-based icons; default inline icon
size is handled by shadcn button styles unless a component
needs an explicit size.
